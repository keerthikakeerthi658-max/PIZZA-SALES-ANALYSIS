Project Overview

This project demonstrates a complete Pizza Sales Analysis workflow, from raw data to interactive insights. It includes the following steps: importing CSV sales data into a SQL database, performing data cleaning and analysis, and creating an interactive dashboard to visualize key sales metrics. The dashboard provides insights into sales trends, popular items, and revenue performance, enabling data-driven decision-making. This project showcases practical skills in SQL, data analysis, and dashboard development, making it a comprehensive example of end-to-end business analytics.
